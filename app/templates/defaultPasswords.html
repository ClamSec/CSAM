Password Strength & Cracking [3/6]

Many devices use default passwords. Manufacturers provide these default credentials to make the initial setup of a device simpler. Default credentials are also used whenever a device is reset back to factory settings.
Default passwords are usually simple, and not unique to each individual device. These default credentials can be found in the instruction manuals, or on the device itself. The expecation is that each user will change their password to something that is unique; however, many do not. This presents an enormous security concern.
